# Interactive Graphic Design Portfolio

## Description
This project is a modern and interactive personal portfolio website created to showcase my graphic design works in a unique and engaging way.

Instead of a traditional portfolio layout, this website is designed as a digital design museum where visitors can explore my projects one by one while scrolling.

The homepage features a 3D cartoon character representing my digital identity with cinematic lighting and glowing background effects.

The website includes smooth animations, cursor interactions, and immersive visual effects to create a memorable experience for visitors.

---

## Features

- Interactive portfolio layout
- 3D character hero section
- Smooth scroll animations
- Cursor-based interactions
- Museum-style design showcase
- Modern UI design

---

## Technologies Used

- HTML
- CSS
- JavaScript
- GSAP
- Three.js
- React / Next.js
